# Installation


## Needed packages

Nerd Fonts used for the icons in the status bar, so you need to install them first.
```
sudo pacman -S nerd-fonts-jetbrains-mono
```

Or download all Nerd Fonts 
```
sudo pacman -S nerd-fonts
```
